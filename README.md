LMS UI Walkthrough
I have implemented a modern, responsive LMS UI using vanilla HTML and CSS. The design features a clean, professional aesthetic with a consistent color palette and layout.

Implemented Pages
1. Login (
index.html
)
Features: Clean centered card layout, email/password inputs, "Remember me" option.
Style: Uses the primary indigo color for the call-to-action.
2. Dashboard (
dashboard.html
)
Features:
Sidebar navigation.
Top header with user profile.
Key stats cards (In Progress, Completed, Average Grade).
"Continue Learning" section with progress bars.
Upcoming deadlines list.
Layout: CSS Grid for the main content area.
3. Courses (
courses.html
)
Features: Grid of course cards with course icons (JS, UX, CSS, PY).
Details: Progress bars indicating completion status.
4. Course Detail (
course-detail.html
)
Features:
Two-column layout: Video/Content on the left, Syllabus on the right.
Collapsible accordion for syllabus modules.
Downloadable resources section.
5. Assignments (
assignments.html
)
Features: Data table listing assignments, due dates, and status badges (Pending, Submitted, Graded).
6. Profile (
profile.html
)
Features: User settings form with inputs for personal info and bio.
